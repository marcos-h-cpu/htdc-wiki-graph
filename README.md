A web-based tool that generates interactive network graphs to visualize Wikipedia "rabbit holes". This kind of exploration can be disorienting and anti-retentive, making it difficult to recall  how one idea led to another. This project is a network graph where each node represents a Wikipedia article, and edges represent the hyperlinks users followed between articles. By focusing on the user’s path this visualization offers a more digestible way of understanding how information branches and connects.
